## [1.0.1] - 2026-06-02

### 🔧 更改
- **配置写入改由后台 Service Worker 处理**
  - 内容脚本不再直接写入 `chrome.storage.local`；所有变更操作（`add-selector`、`remove-selector`、`clear-site`）均以经过验证的 `toc:mutateConfig` 消息发送到后台脚本
  - 后台通过新增的 `shared/config-primitives.js` 模块，使用 `serializedWrite` 安全处理多标签页并发写入
- **Edge Dock 预览视觉优化**
  - 更小、居中对齐的预览横线和设置图标，使用 CSS 自定义属性控制亮/暗色预览颜色
  - 当前项改用环形高亮替代纯色填充；焦点环使用 `box-shadow` 替代 `outline`
- **DOM 监听在默认模式下精确过滤**
  - 未配置自定义选择器时，仅当实际标题元素受影响才触发重建，减少动态页面上的无谓重建

### 🐛 修复
- **徽章模式下内容未变时的闪烁问题** — 当目录项未发生实质变化时跳过 UI 重建，避免频繁 DOM 变动导致的视觉闪烁
- **URL 轮询监听准确性** — 使用 `WeakMap` 元素标识和文本内容哈希进行更可靠的变化检测；MutationObserver 活跃时自动使用节流轮询间隔
- **`uniqueInDocumentOrder` 排序正确性** — 去重后增加基于 `compareDocumentPosition` 的显式排序，确保混合选择器来源的元素按正确 DOM 顺序排列
- **Edge Dock 预览自动收起时机** — 面板打开触发的预览现在在 1.8 秒后自动收起；关闭展开面板时在现代模式下正确收起工具条

### ⚡ 技术改进
- 新增 `shared/config-primitives.js` 模块，提供 `applyTocConfigMutation` — 归一化、校验、限流的配置写入
- `background.js` 处理 `toc:mutateConfig` 消息，包含发送者验证和站点 URL 模式校验
- 内容脚本监听 `tocConfigs` 存储变更，通过 `refreshConfig()` 实现跨标签页配置同步
- `collectBySelector` 支持可选的 `maxCandidates` 参数，用于受控轮询查询

## [1.0.0] - 2026-06-02

### 🚀 新增
- **Edge Dock 吸附式目录工具条**
  - 用独立圆形的插件列表标记和实时目录缩略图替代折叠悬浮按钮

  - 支持点击收起态横线直接跳转，不改变目录卡片的临时展开状态
  - 桌面端悬停后向页面内侧展开，离开后恢复横线；触屏设备支持临时点击切换
  - 快捷设置提供刷新、拾取元素、站点配置和左右侧切换

  - 默认使用新版 Edge Dock
  - 支持切换回经典文字徽章和自由拖拽浮动面板交互
  - 通过 `chrome.storage.local` 将模式切换同步到已打开的标签页

### 🔧 更改
- **目录面板调整为轻量吸附卡片**

  - 保留导航、高亮、无障碍标签和增量更新
  - 移除面板自由拖动；工具条仅上下移动，并保留视口安全边距
- **继续兼容现有位置存储**
  - 复用 `tocBadgePosMap` 保存工具条侧边和竖直锚点

### ⚡ 技术改进
- **当前项跟踪**在目录卡片收起后仍持续运行，滚动页面时同步更新缩略横线高亮
- **经典模式布局**恢复经典结构化标题栏与操作分组，同时保持新增模式切换入口紧凑
- **Edge Dock 测试**覆盖标题层级、缩略横线截取、悬停延迟、纯 hover 状态、触屏激活、视口约束、清理隔离和嵌套面板移除

## [0.8.1] - 2026-05-18

### 🚀 新增
- **MAIN world 脚本用于 SPA 导航安全**
  - 新增 `page-url-hook.js`，注入到页面的 MAIN world 中包装 `history.pushState`/`replaceState`
  - 通过自定义 `toc:urlchange` 事件通知 URL 监控，避免内容脚本直接修改 History API

### 🔧 更改
- **URL 监控改用自定义事件**，不再直接包装 History API 方法
  - 不再从内容脚本上下文覆盖 `history.pushState`/`history.replaceState`
  - 在 ChatGPT 等大量修改 History API 的网站上安全运行
- **防抖参数调优**：DEBOUNCE_MS 500→400ms，MAX_DYNAMIC_DEBOUNCE_MS 1000→1800ms，初始响应更快，高频变更回退更合理
- **构建脚本**现在会复制 `page-url-hook.js` 到 dist，并在打包前删除已有 zip

### 🐛 修复
- **按钮类型安全**：所有动态创建的 `<button>` 元素显式设置 `type="button"`，防止意外触发表单提交

### ⚡ 技术改进
- **导航安全测试套件**：在 `checks/navigation-safety.test.mjs` 中新增 Node.js 测试，覆盖 URL 监控、页面钩子、按钮类型和重建时序

## [0.8.0] - 2026-05-15

### 🚀 新增
- **ES Modules 架构与 esbuild 打包**
  - 内容脚本从单体 `utils.js` 和全局命名空间注入迁移至标准 ESM `import`/`export`
  - esbuild 在构建时将 `src/content.js` 打包为单个 IIFE，无需运行时加载顺序管理
- **构建系统**
  - 新增 `build.js`，集成 esbuild 打包、语法验证和 dist 分发打包
  - 构建产出 `dist/build/`（运行时文件）和 `dist/packages/v{版本号}.zip`
- **模块化架构**
  - 将单体 `utils.js`（1126行）拆分为 `src/utils/` 下 11 个独立工具模块
  - 将 `mutation-observer.js` 拆分为 3 个专注模块：`dom-watcher.js`、`url-monitor.js`、`rebuild-scheduler.js`
  - 新增独立的 `nav-lock.js` 导航锁模块
  - 新增 `event-bus.js` 用于模块间解耦通信
  - 新增 `focus-trap.js` 共享工具
  - 从浮动面板中提取 `floating-panel-helpers.js`
- **URL 变化监测**
  - 通过 History API 拦截（`pushState`/`replaceState`）+ 轮询回退，支持 SPA 页面
- **CSS 自定义属性主题**
  - 用 CSS 自定义属性（`--toc-bg-panel` 等）替代重复的亮色/暗色 CSS 规则
- **CI/CD 发布工作流**

### 🔧 更改
- **模块系统演进**：全局命名空间 → 自定义 `define()`/`require()` → 标准 ESM
- **存储原语**：单一 ESM 源文件；构建时为后台 service worker 产出单独的 IIFE bundle
- **常量整合**：分散的 `uiConst()` 调用整合为各模块内的 `CFG` 对象
- **动态导航锁时长**：根据滚动距离调整锁定时间

### 🐛 修复
- **暗色主题**：未应用到"选择器已生成"和"站点配置"覆盖层
- **重复上下文失效通知**：防止每次重建时重复显示 `ctxInvalidatedNotice`
- **Service Worker**：修复 `importScripts` 路径和双上下文存储原语
- **重建调度器**：修复通过 `handle.disconnect()` 断开连接

### ⚡ 技术改进
- **分层可见性过滤**：三阶段检查并提前终止 — 先做轻量 DOM 检查，再读样式/几何信息，最后检查父元素裁剪
- **O(n) Set 去重**：`uniqueInDocumentOrder` 中用 Set 替代排序去重
- **共享存储原语**：提取为单一源文件，消除上下文间重复

## [0.7.0] - 2026-05-07

### 🚀 新增
- **触摸/触控笔拖拽支持**
  - 拖拽系统从 mouse events 迁移到 pointer events，支持触摸屏和触控笔
  - 折叠徽标添加 `touch-action: none`，防止拖拽时触发页面滚动
- **`data-toc-owner` 命名空间属性**
  - 所有扩展 UI 元素添加 `data-toc-owner="web-toc-assistant"` 属性，避免与宿主页面元素冲突
- **增量面板更新**
  - 新增 `updateItems` 方法，重建时仅更新变化的 TOC 项，避免完整重新渲染
- **写入前日志（write-ahead intent）**
  - 后台 service worker 写入前保存意图日志，防止 MV3 休眠导致数据丢失
- **存储写入序列化**
  - 所有存储写入通过队列序列化，防止 TOCTOU 竞态条件
- **重建熔断器**
  - 追踪连续重建失败次数，超过阈值后冷却，防止无限重建循环
- **上下文失效刷新链接**
  - 扩展上下文失效通知中添加可点击的"刷新页面"链接

### 🔧 更改
- **渲染函数重构为选项对象**
  - `renderFloatingPanel` 从 14 个位置参数改为选项对象参数
- **常量配置整合为 CFG 对象**
  - 所有模块的常量从分散的 `uiConst()` 调用整合为集中的 `CFG` 对象
- **MutationObserver 无条件启动**
  - 不再检查选择器有效性，始终启动观察，改善 SPA 页面支持
- **`pendingRebuild` 改为计数器**
  - 防止高频 DOM 变更时丢失重建请求
- **并行标签处理**
  - `updateIconsForOrigin` 和 `broadcastEnabledToOrigin` 改为 `Promise.allSettled` 并行处理
- **动态防抖参数调优**
  - 指数从 1.5 降至 1.3，上限从 2000ms 降至 1000ms，活跃页面上 TOC 更新更快
- **新增 `alarms` 权限**
  - 使用 `chrome.alarms` 替代 `setInterval` 进行后台定时任务
- **CSS 选择器验证不再执行 DOM 查询**
  - 移除 `querySelector` 回退验证，避免副作用

### 🐛 修复
- 元素拾取器 Escape 键不再折叠 TOC 面板
- **折叠徽标位置漂移**
  - 移除 `expand()` 中的位置保存，防止累积漂移
- 面板头部按钮点击无响应
- 元素拾取器高亮使用 `position:fixed` 修正滚动偏移
- CSS 转义 polyfill 处理连字符后数字
- 浮动面板 roving tabindex、removalObserver、scroll 回退、用户选择追踪
- 后台 service worker 双重切换、双击竞态、alarm 重置
- 配置写入错误处理、空值保护、变更防抖重置
- toc-app 重建竞态、展开闪烁、活动状态恢复
- 内容脚本清理钩子和去重展开/折叠逻辑
- `destroy()` 后重建循环正确停止
- `addWindowListener` 信号监听器防重复注册
- 拖拽取消时不再保存位置
- 后台 service worker 重启后可通过 `__TOC_APP_LOADED__` 重置重新启用

### ⚡ 技术改进
- **批量布局读取**
  - `isElementVisible` 重构为 `batchCollectVisibility`，一次同步布局遍历替代逐元素查询
- **内存缓存**
  - 面板展开状态和徽标位置使用内存缓存，减少存储读取
- **removalObserver 优化**
  - 缩窄为 `document.body` 的 `childList` 监听，替代全子树监听
- **配置脏标记**
  - 重建时仅在配置变更后读取存储，跳过无变更读取
- **模块依赖校验**
  - 所有模块添加加载时依赖检查，缺失时输出明确错误日志
- **IntersectionObserver 条目缓冲**
  - RAF 节流期间缓冲条目而非丢弃，避免丢失滚动位置
- **每选择器匹配上限**
  - CSS `querySelectorAll` 结果添加上限，防止宽泛选择器导致页面冻结
- **i18n 补全**
  - 新增 `ctxInvalidatedNotice`、`ctxInvalidatedRefresh`、`titleStorageQuotaExceeded`、`titleStorageWriteFailed`、`warningStorageQuotaPruned` 等国际化键
