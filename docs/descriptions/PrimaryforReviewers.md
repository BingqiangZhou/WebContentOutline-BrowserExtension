No user account or login is required to test this extension. It functions immediately upon installation.

Testing Instructions:

Navigate to a structured webpage: Visit any long article, documentation page, or a Wikipedia entry (e.g., https://en.wikipedia.org/wiki/Google).

Activate: Click the extension icon in the browser toolbar. The TOC panel will automatically appear on the right side of the webpage.

Interact: Click the button to expand the Table of Contents. Click any list item to scroll to that section.

Settings: You can toggle the "Enable on this site" switch in the extension popup menu. All preferences are stored locally.

Note: The extension uses host permissions solely to scan the DOM for heading tags (h1-h6) to generate the navigation list. No data is sent to remote servers.
